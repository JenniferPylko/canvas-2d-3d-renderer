Canvas 2D 3D Renderer

About This Demo

Using a renderer implented in Canvas2D, this lets you manipulate a couple different periodic functions in 3D. The sin(x) + sin(z) option displays 2 simple intersecting sine waves. The Random Oscillation oscillates every vertex up and down with a random phase offset. None displays a flat plane for comparison.

To display the way vertex colors are interpolated, the material's style can be either a height-dependent grass and dirt, each vertex set to red, green, and blue, or a solid gray. Light Power and Shininess are used to calculate vertex illumination based on the Blinn-Phong reflection model. Per-pixel calculation and surface normals are too computationally expensive to render with this method.

This is NOT a practical way to render in 3D. WebGL will be much faster and support more complex effects.
